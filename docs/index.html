Edit commands
u Undo last command
U Undo all commands for current line
. Repeat last command
y <MC> Copy text depending on movement command
yy , 5yy Copy current line (yank), 5 lines from current
p , P Paste below, above
d <MC> Delete text depending on movement command
D Delete text to the end of line
dd , 5dd delete current line, Delete 5 lines downwards from current
:10,20d Delete lines 10 - 20
x , 5x Delete current character, 5 chars from current
d0 Delete to beginning of line
d$ Delete te end of line
Search commands
<leader> gf (Intellij) Find in Path
<leader> ga (Intellij) Search Everywhere
<leader> gc (Intellij) Goto Class
<leader> gi (Intellij) Goto Implementation
<leader> gd (Intellij) Goto Declaration
<leader> fu (Intellij) Find Usages
-----
/foo Search forwards for "foo"
?foo Search backwards for "foo"
n Repeat search (keep direction)
n Repeat search (reverse direction)
Refactoring
<leader> rf (Intellij) Reformat Code and Save
<leader> rr (Intellij) Rename Class/Function/Variable
<leader> rs (Intellij) Show Intention Actions
<leader> cc (Intellij) Toggle Comment (in VISUAL mode)
Movement commands <MC>
l , [LEFT] Move one character left (lower L)
h , [RIGHT] Move one character right
k , [UP] Move one line upwards
j , [DOWN] Move one line downwards
5 <MC> Move 5 times the command.
w , b Move one word left / right
0 , $ Move to beginning / end of the line
+ , - Move to first character of next / previous line
( , ) Move to next / previous sentence
{ , } Move to next / previous paragraph
% Move to the corresponding opening/closing bracket: ()'s, []'s and {}'s
G Go to last line
5G , :5 Go to line 5
Screen commands <MC>
z , z. , z- Position line with cursor at top / middle / bottom
H Go to top of screen (high)
5H Go to 5th line from top of screen
M Go to middle of screen (middle)
L Go to bottom of screen (low)
v Switch to visual mode (show selection)
